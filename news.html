28. May 2025, 15:47; uploaded by Pys0n

First Elements and Features added
The first new features have been added. You can now enter the world of GoD directly when you start the game instead of going to
the level selection screen. There you can open and close the menu with the ESC key. In the menu you can choose between: Continue game,
Settings and Save and Exit. There are chunks in the world and various elements in the chunks, such as walls, buttons, chests,
collectables and much more.

Next steps: loading the chunks in a certain range, adding the player and running the player.

edited on 13. October 2024, 17:09; uploaded by Pys0n

Version 0.18.0 is now available
The features of version 0.18.0 mainly improve the inventory and skills. Enchanting in the inventory has been revised and many new, small and large, features have been dedicated to enchanting. But also the skills
have been completely renewed, back then everything at a glance, today you can only see the first ones. 65 skills have already been added and a system for navigating through the skill tree. The companions have been
removed, but companion-like creatures will be added again later with a specific character. There are several new attributes for the characters and the rarities of the items have been greatly improved, now not only the
names are colored, because the rarity is now under the name in capital letters and colored, in addition, the name of an item can change when the rarity changes. In the item info, which is also in the inventory, the item
names will now always be slightly larger and bold. In addition, the attributes are changed by the rarity. And 0.19.0 will be released in just one month...
