Version 0.18.0 is now available
The features of version 0.18.0 mainly improve the inventory and skills. Enchanting in the inventory has been revised and many new, small and large, features have been dedicated to enchanting. But also the skills have been completely renewed, back then everything at a glance, today you can only see the first ones. 65 skills have already been added and a system for navigating through the skill tree. The companions have been removed, but companion-like creatures will be added again later with a specific character. There are several new attributes for the characters and the rarities of the items have been greatly improved, now not only the names are colored, because the rarity is now under the name in capital letters and colored, in addition, the name of an item can change when the rarity changes. In the item info, which is also in the inventory, the item names will now always be slightly larger and bold. In addition, the attributes are changed by the rarity. And 0.19.0 will be released in just one month...
